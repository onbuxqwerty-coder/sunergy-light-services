# Sunergy Light Services - Solar Solutions in Ota, Nigeria ☀️🇳🇬

Official landing page for **Sunergy Light Services**, a premier solar energy provider based in Ota, Ogun State. We empower Nigerian homes and businesses with reliable, sustainable, and affordable solar power systems.

## 🚀 Project Overview
This is a high-performance, mobile-first landing page designed to showcase solar products (panels, inverters, batteries) and convert visitors into clients via direct WhatsApp integration.

## 🛠️ Tech Stack
- **React.js** - UI Framework
- **Tailwind CSS** - Styling
- **Lucide React** - Iconography
- **Mobile-Responsive Design**

## 🔋 Key Features
- **Product Packages:** Tiered solutions from basic lighting to full industrial power.
- **Local Focus:** Specifically tailored for the Ota/Nigerian market (fuel-to-solar comparison).
- **Direct Conversion:** Seamless WhatsApp integration for instant quotes.
- **Education:** Built-in FAQ and Lithium vs. Gel battery comparison.
